what is rocketpack

rocketpack is a BaaS (Backend as a Service) cloud infrastructure for developers who need to cover their backend needs. This tool includes the services described below in the features section. With the use of one or more of these services, you don't have to write a backend software for your frontend website or mobile app, and you don't have to run a server and spend a lot of money. Just stick to your frontend app and you can easily use the complicated backend services in a fast and secure cloud environment.

rocketpack Services

data service

Its actually your database. You can use this service to store your data in a fast and secure cloud based no-sql database.

authentication service

Its a tool to manage users for your application with highest security levels

function service

Its a tool to take the logic out of your application and put it in the cloud because of reducing application update versions or increase the calculation performance of the logic

installation

$ npm install rocketpack.io --save

initialization

import rocketpack from 'rocketpack.io'
rocketpack.init({ key: 'YOUR-PROJECT-KEY' })

using data service

const peopleCollection = rocketpack.dataService().collection('people')
let peopleItems = peopleCollection.find()
